Learning Management System (LMS)

API Endpoints

GET
/api/course

Returns all the courses

/api/course/:id

Returns an individual course

/api/course/user/:id

Returns courses by user id

/api/course/curricula/:id

Returns courses by curricula id

/api/course/completed/user/:id

Returns courses by user id that have a completed status

/api/course/:id/:status

Returns courses by user id that have a completed status of 'Not Started' or 'Completed'

e.g:- https://nc-be-northcoders-news.herokuapp.com/api/course/3/Not Started

/api/curricula/

Returns all the curricula

/api/curricula/:id

Returns an individual curricula

/api/curricula/user/:id

Returns curricula by user id

/api/session/

Returns all the sessions

/api/session/:id

Returns an individual session

/api/session/user/:id

Returns sessions by user id

/api/session/user/:id/:status

Returns sessions by user id that have a completed status of 'Not Started' or 'Completed'

e.g:- https://nc-be-northcoders-news.herokuapp.com/api/session/3/Completed

/api/user/

Returns all the users

/api/user/:id

Returns an individual user

/api/username/:username

Returns an individual user by username

Post
/api/course

Add a new course

/api/curricula

Add a new curricula

/api/session

Add a new session

/api/user

Add a new user

Delete
/api/course

Removes a course

/api/curricula

Removes a curricula

/api/session

Removes a session

/api/user

Removes a user

Put
/api/session/:id

Changes a session status from 'Not Started' to 'complete'
